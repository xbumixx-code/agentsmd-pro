Questions

What is AGENTS.md and why do I need it?

AGENTS.md is an instruction file in the project root that explains to an AI agent (Claude Code, Cursor, Copilot) how to work with the codebase. Without it the agent guesses the stack, patterns, and rules. With it — it works like an experienced team member.

Can I use it without signing up?

Yes. Bring your own API key (Claude or OpenAI) and generate unlimited files. No login required.

What's the difference between Simple and Orchestrated?

Simple mode generates one AGENTS.md file with 12 sections (15 sec). Orchestrated mode generates a full multi-agent system: main AGENTS.md + sub-agent files (planner, coder, reviewer, tester) + shared memory files — all in a ZIP archive (30 sec).

What languages and frameworks does it support?

Any. The generator understands descriptions in Russian and English and adapts to any stack: Python, JavaScript, Go, Rust, mobile, data, DevOps. Rules are quantified per language (Python files ≤500 lines, JS ≤300, etc.)
